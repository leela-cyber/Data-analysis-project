Registered Companies EDA
Project Overview

This project performs exploratory data analysis (EDA) on a large dataset of registered companies. The analysis focuses on data inspection, handling missing values, and generating descriptive statistics to understand corporate registration patterns.

Key Steps

Dataset loading and structural inspection

Null value analysis

Data cleaning by removing incomplete records

Descriptive statistical summary

Dataset Size

Over 100,000 company records

17 columns including corporate identifiers, capital details, and registration information

Technologies Used

Python

Pandas
